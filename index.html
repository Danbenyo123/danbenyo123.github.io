Positioning

There are four types of positioning in CSS:

  • Static
  • Relative
  • Absolute
  • Fixed

Static

Static is the default positioning in CSS. It means that the element will be positioned according to the normal flow of the document. This is the default value.

For example:

position:static;

Static

Relative

Relative positioning is the same as static, but the element can be moved from its normal position. The element will still occupy the same space as before, but it can be moved around.

For example:

position:relative;
top:20px;
left:20px;

Relative

Absolute

Absolute positioning is relative to the nearest positioned ancestor. If there is no positioned ancestor, it will be relative to the initial containing block. The element will be removed from the normal flow of the document, and will not affect the position of other elements.

For example:

position:absolute;
top:20px;
left:20px;

Absolute

Fixed

Fixed positioning is relative to the viewport. The element will be removed from the normal flow of the document, and will not affect the position of other elements. The element will remain in the same position even if the page is scrolled.

For example: (look for it in the top right of viewport)

position:fixed;
top:20px;
left:20px;

Sticky

Sticky positioning is a hybrid of relative and fixed positioning. The element is treated as relative positioned until it crosses a specified threshold, at which point it is treated as fixed positioned.

For example:

position:sticky;
top:20px;
left:20px;

Sticky

Overlapping Elements

When elements overlap, the element that appears later in the HTML code will be shown on top. You can use the z-index property to specify the stack order of the elements.

for example:

position: relative;
background-color: blue;
opacity: 90%;
right: 65px;
z-index: 1;

Centering Elements

To center an element horizontally, you can use the margin property with the value auto. To center an element vertically, you can use the position property with the value absolute and the top and bottom properties set to 0.
